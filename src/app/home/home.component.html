Smart Trash Can

In this academic project I developed an Internet of Things solution to make cans smart.

I used several sensors like: ultrasonic sensor, level sensor, servo motor. As a programming card I used Raspberry Pi 3 Model B because it has a Wi-Fi connection to send the sensor data to a remote database. The bin opens automatically by passing his hand in front of the ultrasonic sensor.

I developed two Android applications one for the user and the other for the municipality :

User Application: Allows reporting if there is a failure in a trash, knowing the location of the nearest trash ... Application of municipality : See the problems of trash cans, the shortest way to a trash, Notify if a trash is full ...
